# DungeonCrawler_NoVR

A non-VR dungeon crawler inspired by *Dungeons of Eternity*.

## Features
- First-person movement
- Procedural dungeon rooms
- Melee combat
- Enemy AI with NavMesh

## Controls
- WASD – Move
- Mouse – Look
- Left Click – Attack
- Space – Jump

## Unity Version
2022 LTS or newer
